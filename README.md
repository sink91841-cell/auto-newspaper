# 自媒体报刊抓取工具

一个功能完整的自动化报纸抓取和AI分析工具，支持下载国内外主流报纸并使用AI技术解析其内容。

## 功能特点

### 📰 核心功能
- **多报纸支持**：人民日报（PDF）、纽约时报（图片）
- **智能日期选择**：支持今天、昨天、前天和自定义日期
- **AI内容解析**：使用通义千问多模态模型分析内容
- **纽约时报翻译**：自动将英文内容翻译成中文
- **多格式保存**：支持保存到文件和数据库
- **智能代理检测**：自动适配VPN环境

### 🔧 技术特性
- **模块化设计**：代码结构清晰，易于维护
- **完善的错误处理**：详细的错误提示和解决方案
- **自动数据库管理**：自动创建数据库和数据表
- **安全配置**：敏感信息存储在.env文件
- **实时进度反馈**：下载和解析过程中的状态更新

## 快速开始

### 1. 环境准备

#### 前提条件
- Python 3.8 或更高版本
- 通义千问API Key（用于AI解析）
- PostgreSQL 数据库（可选，用于存储解析结果）
- VPN（用于访问纽约时报，可选）

#### 安装步骤

1. **克隆或下载项目**
   将项目文件保存到本地目录。

2. **运行启动脚本**
   双击 `run.bat` 文件，脚本会自动：
   - 检查Python环境
   - 安装所需依赖
   - 创建默认配置文件
   - 启动程序

3. **配置API Key**
   首次运行时，会创建 `.env` 文件，需要：
   - 打开 `.env` 文件
   - 替换 `TONGYI_API_KEY` 为你的通义千问API Key
   - （可选）配置数据库密码和代理设置

### 2. 使用方法

1. **启动程序**
   双击 `run.bat` 或在命令行运行：
   ```bash
   python main.py
   ```

2. **选择日期**
   - 输入 `1` 选择今天
   - 输入 `2` 选择昨天
   - 输入 `3` 选择前天
   - 输入 `4` 选择自定义日期

3. **选择报纸**
   - 输入 `1` 选择人民日报
   - 输入 `2` 选择纽约时报

4. **等待下载**
   程序会自动下载报纸文件并显示进度。

5. **AI解析**
   - 输入 `y` 开始AI解析
   - 输入 `n` 跳过AI解析

6. **保存结果**
   - 选择是否保存到文件
   - 选择是否保存到数据库

## 配置说明

### 主要配置项（.env文件）

#### 1. API配置
```env
# 通义千问API Key（必填）
TONGYI_API_KEY=your-dashscope-api-key
```

#### 2. 代理配置（可选）
```env
# 代理服务器设置（用于访问纽约时报）
HTTP_PROXY=http://127.0.0.1:7890
HTTPS_PROXY=http://127.0.0.1:7890
```

#### 3. 数据库配置（可选）
```env
# PostgreSQL数据库设置
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=your-database-password
DB_NAME=newspaper_db
```

## 常见问题

### 1. 纽约时报下载失败

**原因**：
- VPN连接不稳定
- 代理配置错误
- 网络防火墙阻止

**解决方案**：
- 检查VPN连接是否正常
- 在 `.env` 文件中配置代理
- 运行 `network_diagnose.py` 诊断网络问题
- 尝试更换VPN服务器

### 2. AI解析失败

**原因**：
- API Key配置错误
- 内容审核不通过
- 网络连接问题

**解决方案**：
- 检查API Key是否正确
- 确保网络连接正常
- 稍后再试，可能是临时问题

### 3. 数据库连接失败

**原因**：
- PostgreSQL服务未启动
- 数据库密码错误
- 数据库不存在

**解决方案**：
- 启动PostgreSQL服务
- 检查数据库密码配置
- 程序会自动创建数据库

## 技术栈

- **Python 3.8+** - 主要开发语言
- **requests** - 网络请求
- **Pillow** - 图片处理
- **pdf2image** - PDF转图片
- **OpenAI SDK** - AI调用
- **psycopg2** - 数据库连接
- **Python-dotenv** - 环境配置

## 注意事项

1. **API Key安全**：不要将包含API Key的 `.env` 文件提交到版本控制系统
2. **网络合规**：使用VPN访问纽约时报时，请遵守相关法律法规
3. **资源使用**：AI解析会消耗API调用次数，请合理使用
4. **文件存储**：报纸文件和解析结果会占用存储空间，请定期清理

## 故障排除

### 网络诊断
运行网络诊断工具检查连接问题：
```bash
python network_diagnose.py
```
