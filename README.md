# 自媒体报刊抓取工具的web功能

### 🔧 技术特性

- **Web界面**：现代化的Web界面，提供直观的操作体验
- **定时任务系统**：支持设置定时抓取任务
- **关键词管理**：支持关键词和分类管理
- **新闻分析**：提供高频关键词统计和相关新闻查询

#### 定时任务
可以使用系统定时任务工具（如Windows任务计划程序）定期运行程序，实现自动抓取和分析。

## Web界面

本项目提供了一个现代化的Web界面，使操作更加直观和便捷。

### 启动Web界面

1. **运行Web服务**
   双击 `launch_web.py` 或在命令行运行：
   ```bash
   python launch_web.py
   ```

2. **访问界面**
   打开浏览器，访问：`http://localhost:5000`

### Web界面功能

#### 1. 任务管理
- 创建和管理报纸抓取任务
- 实时查看任务执行进度
- 支持暂停和停止任务
- 查看任务执行结果

#### 2. 定时任务系统
- 设置定时抓取任务
- 配置执行时间和功能选项
- 管理和监控定时任务
- 查看任务执行历史和日志

#### 3. 历史记录管理
- 查看历史抓取记录
- 下载历史文件
- 删除不需要的记录

#### 4. 关键词管理
- 添加、编辑、删除关键词
- 管理关键词分类
- 导入导出关键词（CSV格式）
- 搜索和筛选关键词

#### 5. 新闻分类
- 基于关键词自动分类新闻内容
- 手动分类调整
- 查看分类统计

#### 6. 高级搜索
- 按日期范围搜索
- 按分类筛选
- 按报纸来源筛选
- 关键词搜索（支持精确匹配）
- 标题关键词搜索

#### 7. 新闻分析
- 高频关键词统计
- 相关新闻查询
- 关键词词云可视化
- 历史新闻分析

### 技术特性

- **响应式设计**：适配不同屏幕尺寸
- **后台任务处理**：任务在后台运行，不阻塞界面
- **实时进度反馈**：任务执行过程中实时更新
- **完善的错误处理**：提供详细的错误信息
- **RESTful API接口**：支持与其他系统集成

### 注意事项

- Web服务默认运行在 `http://localhost:5000`
- 确保端口5000未被其他程序占用
- Web界面需要保持运行状态才能使用
- 关闭Web服务后，定时任务也会停止


## 技术栈

- **Python 3.8+** - 主要开发语言
- **Flask** - Web框架
- **requests** - 网络请求
- **Pillow** - 图片处理
- **pdf2image** - PDF转图片
- **OpenAI SDK** - AI调用
- **psycopg2** - 数据库连接
- **Python-dotenv** - 环境配置
- **HTML/CSS/JavaScript** - 前端技术
